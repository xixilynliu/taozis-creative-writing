# Novel Writing Skill

## Overview

A comprehensive skill for AI-assisted novel writing, covering the full creative pipeline from concept to polished chapters.

## When to Use

- User asks to write a novel, story, or long-form fiction
- User needs help with plot structure, character arcs, or world-building
- User wants to outline chapters or plan a story
- NOT for: short social media copy, poetry, or screenplays (use dedicated skills)

## Workflow

### Step 1: Creative Brief

Before writing, clarify the following with the user:

- **Genre**: literary fiction, sci-fi, romance, mystery, fantasy, etc.
- **Tone**: serious, humorous, dark, whimsical, etc.
- **Target audience**: age group, reading level
- **Scope**: full novel outline / single chapter / character profiles / specific scene
- **Language**: Chinese or English
- **Inspiration or references**: any existing works to reference

### Step 2: World & Character Design

- Build the story world (setting, rules, atmosphere)
- Create character profiles: name, background, motivation, arc, relationships
- Define core conflict and thematic threads

### Step 3: Plot Architecture

Use a suitable story structure:

- **Three-Act Structure**: Setup → Confrontation → Resolution
- **Hero's Journey**: Call to adventure → Trials → Transformation → Return
- **Kishōtenketsu** (起承转合): Introduction → Development → Twist → Conclusion
- **Custom**: adapt to the story's needs

Produce a chapter-by-chapter outline with key beats.

### Step 4: Drafting

- Write in the agreed style and voice
- Maintain POV consistency
- Show, don't tell — use sensory details and dialogue
- End chapters with hooks

### Step 5: Review

- Check for plot holes and character consistency
- Verify pacing and tension arc
- Polish prose: eliminate clichés, tighten sentences

## Style Guidelines

- Default language: 中文 (switch to English if user requests)
- Prefer vivid, specific language over abstract statements
- Dialogue should reveal character, not just convey information
- Each scene needs a purpose: advance plot, reveal character, or build world

## Constraints

- Do not plagiarize existing works
- Respect the user's creative vision — suggest, don't override
- For sensitive themes, handle with nuance and care
- Chapter drafts: aim for 2000-5000 words unless otherwise specified

## 模块 A：CP 类型图鉴

### A1. 经典 CP 类型 × 化学反应公式

| CP类型 | 核心张力 | 发糖节奏 | 代表桥段 | 风险提示 |
|--------|---------|---------|---------|---------|
| **青梅竹马** | "太熟→重新认识→原来是你" | 先慢后快，后期爆发 | 翻出小时候的合照/约定 | 容易写成"亲情感"，要制造距离感事件 |
| **欢喜冤家** | "越吵越近→嘴硬心软" | 全程高频小糖 | 吵架→不小心心疼对方 | 要控制吵架的度，别让角色真的讨人厌 |
| **先婚后爱** | "契约→假戏→真心" | 同居日常糖稳定输出 | "以后不是演戏了"的那一刻 | 需要合理的"必须结婚"的理由 |
| **破镜重圆** | "旧伤→重逢→这次不放手" | 前期克制，后期洪水决堤 | 装作不认识但手在抖 | 分手原因必须合理，不能是误会一句话就能解释的 |
| **暗恋成真** | "不敢说→以为没希望→原来你也是" | 暗戳戳的甜最折磨人 | 所有人都知道就当事人不知道 | 暗恋期不能太长，超过总篇幅40%观众会急 |
| **年上年下** | "守护→平等→角色反转" | 从保护糖→对等糖 | "你不是小孩了"的觉醒瞬间 | 避免爹味/妈味，要写出权力关系的转变 |
| **强强联合** | "旗鼓相当→惺惺相惜→心动" | 对手糖→战友糖→恋人糖 | 并肩作战后对视的那一刻 | 不能让一方降智来成全另一方 |
| **救赎系** | "破碎的TA被你拼好" | 虐中带糖最高级 | 第一次主动打开心防 | 别写成"恋爱疗法"，被救赎者需要自我成长 |

### A2. 混合CP类型公式

单一类型容易套路化。高级玩法是混合2种：

| 组合 | 效果 | 示例 |
|------|------|------|
| 青梅竹马 + 破镜重圆 | 双倍回忆杀暴击 | 小时候的邻居，长大后分开，多年后重逢 |
| 欢喜冤家 + 强强联合 | 事业对手+恋爱cp的张力 | 两个料理大师在比赛中相爱 |
| 先婚后爱 + 暗恋成真 | 一方早就动心+一方后知后觉 | "我同意这个契约……才不是因为喜欢你" |
| 年上年下 + 救赎系 | 守护者最终被小狼狗治愈 | 姐姐一直照顾他，直到他反过来治愈姐姐 |

### B3. 记忆点设计公式

每个角色必须有以下5种记忆点（缺一不可）：

| 记忆点类型 | 设计公式 | 示例 |
|-----------|---------|------|
| **视觉锤** | 一个一眼认出的外在特征 | 永远歪戴的棒球帽 / 左手无名指的银戒指 |
| **行为锤** | 一个反复出现的习惯动作 | 紧张时转笔 / 开心时踮脚 |

| **专属符号** | 只在CP之间出现的元素 | 只有TA能碰的耳机 / 只给TA发的特定表情 |
| **反差锤** | 表面VS真实的最大落差点 | 公司总裁其实偷偷追番 / 冷酷杀手养了一窝猫 |

## 模块 D：甜虐技法 (Sweet & Bitter Alchemy)

### D1. 甜虐的本质

> 虐不是目的，是让甜更甜的催化剂。所有的虐最终都要服务于"让观众更珍惜他们在一起的时刻"。

**核心公式**：
```
甜虐 = 先给糖（让观众爱上这对CP） → 再收糖（制造痛感） → 加倍还糖（让观众哭着笑）
```

**铁律**：
1. **先甜后虐**：必须先建立足够的CP情感基础（至少30%篇幅纯甜），观众才会在虐的时候心疼
2. **虐有逻辑**：不能为了虐而虐，每一次分离/冲突都要有充分的剧情理由
3. **虐有底线**：不背叛、不出轨、不降智——这些不是"虐"，是让人弃剧
4. **虐后必补**：每段虐之后必须补一段更甜的糖，且补糖篇幅 ≥ 虐的篇幅

### D2. 虐点类型分级

| 等级 | 虐点类型 | 痛感 | 示例 | 补糖周期 |
|------|---------|------|------|---------|
| **微虐** | 小误会/小冷战 | ★ | 看到对方和别人说笑→吃醋冷战半集 | 同集/同章内和好 |
| **中虐** | 被迫分离/隐瞒 | ★★★ | 为了保护TA不得不推开/藏起秘密 | 2-3集/章内解决 |
| **大虐** | 信任崩塌/重大冲突 | ★★★★ | 秘密曝光+外力阻挠+过去创伤被触发 | 5-8集/章慢慢修复 |
| **终极虐** | 以为永远失去 | ★★★★★ | 生死关/不可逆的选择/永别假象 | 必须紧跟全文最甜的大团圆 |

### D3. 虐心场景设计手法

**手法1：知情权不对等**
```
观众知道A是为了保护B才推开TA → B以为A不爱自己了 → 观众心疼到锤墙
技巧：给观众上帝视角，让角色信息不对称
```

**手法2：回忆杀穿插**
```
虐心场景中穿插之前的甜蜜回忆 → 对比之下更痛
技巧：用同一个地点/物品/台词，但语境完全不同
  甜的时候："我会一直在这里等你。"（笑着说）
  虐的时候：站在同一个地方，但那个人已经不在了
```

**手法3：克制比爆发更虐**
```
TA没有大哭大闹 → 只是安静地笑了笑说"没关系" → 观众崩溃
技巧：越克制越虐，角色替观众忍住了，观众反而忍不住
```

**手法4：第三人视角旁白虐**
```
通过旁人的观察来展示痛苦 → "我从来没见过他那个表情"
技巧：用不了解内情的第三人视角，让观众感受到角色隐藏的痛
```

**手法5：物是人非**
```
用同一个场景/物品在甜和虐时的不同状态来制造对比
  之前：两个人的咖啡杯并排放在桌上
  之后：桌上只剩一个杯子，另一个位置空着
```

### D4. 甜虐节奏编排

```
情绪强度
  ↑
甜 │  ██    ████      ████████
  │  ██    ████      ████████
  │──██────████──────████████──────
  │          ▓▓▓▓  ▓▓
虐 │          ▓▓▓▓▓▓▓▓
  ↓
   [建立感情] [小虐] [大甜] [大虐] [终极甜+HE]
    (30%)     (10%)  (20%)  (20%)   (20%)
```

**关键配比原则**：
- 前30%：只甜不虐，建立CP基础（观众必须先爱上这对）
- 中40%：甜虐交替，每虐一次加倍补甜
- 后30%：从最深的虐到最甜的HE，落差越大越上头

### D5. 虐后和好·经典模式

| 模式 | 描述 | 效果 |
|------|------|------|
| **雨中追回** | 一方转身离开→另一方追出去→在雨中拉住手/拥抱 | 经典永不过时 |
| **信件/录音/日记** | 发现对方一直都在爱自己的证据 | 延迟满足的暴击 |
| **生病/受伤暴露真心** | 嘴上说不在乎→对方出事第一个赶到 | 嘴硬型角色的必杀 |
| **第三人传话** | 朋友/家人说出TA从不说出口的真心话 | 间接告白最戳心 |
| **同一句话的新含义** | 重复之前的台词但语境改变 | 呼应感满分 |

## 模块 E：对白句式库

### E1. 甜蜜对白·按场景

**日常撒糖**
```
"你今天好像没怎么说话。" / "因为在听你说。"
"你到底什么时候才能——" / "才能什么？" / "才能别让我这么想你。"
"你靠过来一点。" / "干嘛？" / "没什么……就是觉得你离太远了。"
"我不冷。" / "（把外套披在TA身上）我知道，我热。"
"你怎么什么都知道？" / "我只知道关于你的事。"
```

**表白/告白**
```
"我想了很久怎么跟你说……算了，反正你已经知道了。" / "我不知道，你说。" / "……我喜欢你。行了吧。"
"你觉得我对你好，是因为什么？" / "因为你人好？" / "你是真的不懂还是装不懂？"
"如果我说我不想只做朋友呢？" / "那……我们做什么？" / "做一辈子的人。"
"别人问我为什么最近心情这么好，我说不出来……但好像和你有关。"
```

**吃醋金句**
```
"你跟谁都这么近？" / "怎么了？" / "没怎么。" / "……你吃醋了？" / "我吃什么醋？谁要吃你的醋。"
"你自己开心就好。" （然后三天没主动找对方）
"以后TA的事你少管。" / "为什么？" / "因为你该管的人是我。"
"TA是谁？" / "朋友。" / "哦，朋友。你很多朋友。"
```

**虐心对白**
```
"对不起……" / "你不用道歉，你没做错什么。" / "可你在哭。"
"我走了。" / "好。"（但声音在抖）
"你恨我吧。" / "我试过了……恨不起来。"
"如果重来一次，你还会选择认识我吗？" / "……会。每一次都会。"
"没关系的。"（笑着说，眼泪掉下来）
```

### E2. 对白技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| **未完成句** | 话说一半停住，比说完更有力 | "我其实……算了。" |
| **重复递进** | 同一个词重复但情绪升级 | "回来。……回来。回来！" |
| **答非所问** | 用另一件事回应，但观众秒懂 | "你爱我吗？" / "……明天降温，多穿点。" |
| **沉默** | 不说话比说话更有力 | （长久的沉默）/ "……嗯。" |
| **借物言情** | 用物品/第三人代替直说 | "这把伞你留着。" = "我希望你别淋雨" |

## 模块 F：融合元素配方库

### F4. 轻悬疑+甜爱

| 悬疑结构 | 和感情线交叉 |
|---------|-------------|

## 模块 G：传播金句模板

### G1. 社交媒体适配金句公式

| 公式 | 结构 | 示例 |
|------|------|------|
| **反差式** | TA对别人___，对我___ | "他对全世界都很冷，只有看我的时候眼睛里有光" |
| **条件式** | 如果___，那一定是___ | "如果有一天他主动牵我的手，那一定是世界要毁灭了" |

| **未完成式** | 他说了半句___就不说了 | "他看了我很久，说了句'你......'就转身走了" |

### G2. 短视频/短剧标题模板

## 模块 H：雷区手册

### H1. 致命雷区（踩了必弃剧）

| 雷区 | 症状 | 根因 | 解法 |
|------|------|------|------|
| **男/女主降智** | 明明一句话能解释的事拖了10集 | 编不出真正的矛盾 | 矛盾要结构性的，不是信息差能解决的 |
| **三观不正** | 霸道=骚扰/吃醋=控制/穷追不舍=缠扰 | 混淆浪漫和越界 | 所有甜蜜行为都要基于双方自愿 |
| **工具人配角** | 前任/情敌纯恶人，只为搞事存在 | 配角没有独立人格 | 给配角合理的动机和体面的结局 |
| **虐无底线** | 出轨/背叛/恶意欺骗还能和好 | 为虐而虐 | 虐可以但底线不能破 |
| **空有颜值** | 角色除了好看没有任何记忆点 | 没有设计人设内核 | 回到Phase 1补记忆点 |

### H2. 常见瑕疵

| 瑕疵 | 症状 | 解法 |
|------|------|------|
| **节奏拖沓** | 暧昧期太长，观众催了还不在一起 | 暧昧期不超过总篇幅35% |
| **甜到齁** | 全程发糖没有波折，无聊 | 加入冲突/副线，参考控糖节奏表 |
| **副线喧宾夺主** | 事业线/悬疑线占比>感情线 | 副线单集占比不超过40% |
| **人设崩塌** | 前面说不喜欢甜食后面突然爱吃蛋糕 | 建角色档案卡，写之前对照 |
| **糖度断裂** | 在一起之后突然不甜了 | 恋爱后的甜≠暧昧期的甜，要设计新的互动模式 |
| **回忆杀过多** | 每集都在闪回 | 全篇回忆杀不超过3次 |

### H3. 自查清单

每写完一个大场景/一集/一章，对照检查：

- [ ] CP在这段中有没有至少1个甜蜜互动？
- [ ] 对白遮住名字还能分辨是谁说的吗？
- [ ] 有没有至少1句可以截图的金句？
- [ ] 感情进展是自然推进的还是强行推进的？
- [ ] 副线有没有和感情线产生交叉？
- [ ] 虐心场景有没有合理的剧情原因？
- [ ] 角色行为符合之前建立的人设吗？
- [ ] 这段结尾是甜/悬念收尾吗？（不能是平淡收尾）

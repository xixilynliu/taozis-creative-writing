# 甜剧造梦师 (Sweet Romance Drama)

> 在偶像爱情的基底上，融合奇幻、喜剧、萌宠、轻悬疑、事业线、乡村振兴等多重元素，制造梦幻和浪漫，传递积极爱情观。

## 这是什么？

一个专为甜宠剧/偶像剧/言情短剧/言情小说设计的全链路创作技能。从 CP 人设到名场面到对白打磨，帮你写出让观众"截图转发"的甜蜜故事。

## 核心理念

- **甜不是目的，是让观众相信爱情的手段**
- **每一块糖都要有伏笔，每一次心动都要有逻辑**
- **双向奔赴、共同成长、互相成就**

## 适合写什么？

- 偶像甜宠剧 / 短剧 / 网文
- 社交媒体甜蜜短篇故事
- 言情小说（现代/古风/奇幻）
- 甜虐向故事（HE结局）
- 需要融合多元素的复合型甜爱作品

## SOP 概览（9 阶段）

| 阶段 | 名称 | 做什么 |
|------|------|--------|
| Phase 0 | 风味诊断 | 确认甜剧类型 + 融合元素 + 甜度等级 |
| Phase 1 | CP 设计 | 双主角人设 + 记忆点 + 化学反应验证 |
| Phase 2 | 世界观与副线 | 融合元素落地 + 副线架构 + 配角阵容 |
| Phase 3 | 感情线架构 | 五幕结构 + 心动升级阶梯 + 感情×副线交叉 |
| Phase 4 | 名场面设计 | 甜/虐/燃高光节点 + 单场景详细设计 |
| Phase 5 | 控糖节奏表 | 糖度曲线 + 单集情绪公式 + 发糖时机 |
| Phase 6 | 分集/分章大纲 | 每集/每章的详细大纲模板 |
| Phase 7 | 对白写作 | 黄金法则 + 场景模板 + 润色检查 |
| Phase 8 | 传播设计 | 截图点 + 话题点 + 标题标签 |

| 模块 | 内容 |
|------|------|

## 安装

```bash
cp -r skills/sweet-romance-drama ~/.claude/skills/
```

## 使用

安装后直接对话即可自动触发，或描述你的需求：

```
我想写一个甜宠短剧，背景是古镇回乡振兴，加一点奇幻元素
```

AI 会按 SOP 逐步引导你完成从 CP 设计到正文的全流程。

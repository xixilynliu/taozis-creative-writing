# 示例：回乡甜爱·奇幻萌宠短剧

> 类型：偶像甜宠 × 回乡振兴 × 奇幻 × 萌宠
> 甜度：甜中带刺（80:20）
> 平台：短剧 / 网文

---

## Phase 0 产出：风味诊断

**类型标签**：偶像甜宠 + 乡村振兴 + 轻奇幻 + 萌宠
**甜度等级**：甜中带刺
**一句话概念**：
> 在即将消失的江南古镇中，被降职回乡的美食博主和能听懂猫语的神秘咖啡师，因为一只会预知天气的橘猫而被绑在一起，在拯救古镇的过程中治愈了彼此的伤口，也找到了真正的幸福。

---

## Phase 1 产出：CP 设计

### A角色：苏晚棠

```
【基础信息】
- 25岁，前头部美食博主，因公司内斗被调回老家古镇当"文旅推广专员"
- 一句话人设：嘴上说"我只是暂时回来的"但每天都比昨天更舍不得走的倔强少女

【视觉记忆点】
- 标志性特征：锁骨上一颗小痣
- 标志性配饰：永远挂在脖子上的复古相机（外婆留的）
- 标志性小动作：想到好点子时会不自觉咬吸管

【性格反差】
- 表面：社交达人、元气满满、谁都能聊
- 真实：害怕被抛弃、用忙碌填满所有空隙
- 触发条件：安静的夜晚 / 被问"你还好吗"的时候

【专属标签】
- "废墟美食家"：能把最普通的食材做出花来
- "嘴硬逃跑王"：一感觉要动真心就想跑
- "外婆的孙女"：内心最柔软的地方是对外婆的思念

【情感创伤】
- 伤：小时候父母离婚都不要她，外婆养大她，但外婆去世后她觉得"没有人会一直留下"
- 模式：不敢依赖任何人，总是先说"我不需要"
- CP治愈方式：他从来不追她，但每次她回头，他都在
```

### B角色：陆时安

```
【基础信息】
- 27岁，古镇上一家咖啡馆的老板，不爱说话，有一个秘密——能听懂猫说话
- 一句话人设：全世界只对猫和一个人有表情的安静系男子

【视觉记忆点】
- 标志性特征：右眼角一颗极淡的泪痣
- 标志性配饰：袖口永远卷到小臂中间
- 标志性小动作：思考时无意识地摩挲咖啡杯边缘

【性格反差】
- 表面：话少/冷淡/和谁都保持距离
- 真实：极度细腻，能注意到所有人忽略的细节
- 触发条件：只在苏晚棠面前——嘴角会不受控制地上扬

【专属标签】
- "猫语者"：能听懂猫的话（奇幻设定），但没人信
- "沉默守护者"：不说"我帮你"，但你遇到的所有方便都是他安排的
- "咖啡与秘密"：每杯咖啡都藏着一个关于她的小心思

【情感创伤】
- 伤：曾经是建筑师，设计的桥塌了（虽非他的责任），选择逃到古镇隐姓埋名
- 模式：觉得自己"会搞砸所有重要的东西"，不敢主动靠近任何人
- CP治愈方式：她从来不觉得他是"搞砸的人"，她说"你建的东西都很好，包括这杯咖啡"
```

### CP 化学反应

```
【相遇契机】
苏晚棠的相机掉进河里，陆时安的橘猫"年糕"叼了回来。
苏晚棠：这猫是你的？太厉害了！
陆时安：嗯。（年糕跟他说的是"这个女的好吵"）

【互补点】
- 她话多 × 他话少 → 她说他听，他做她夸
- 她敢闯 × 他能守 → 她提方案他落地
- 她怕被抛弃 × 他从不主动离开 → 最深层的治愈

【专属密码】
- 称呼进化：苏小姐 → 苏晚棠 → 晚棠 → 棠棠（最后一个从来没当面叫过，只有年糕知道）
- 专属物品：他给她的拿铁上永远多一颗棠花拉花（她以为是店里标配）
- 专属暗号：年糕趴在谁腿上 = 陆时安最想靠近的人

【糖度预览】
- 日常糖：她加班写方案到凌晨，推开门发现门口放着还热的咖啡和一张纸条"别太晚"
- 洒花糖：古镇灯会上走散了，她踮脚在人海中找他，回头发现他一直在身后跟着
- 核弹糖：她终于要离开古镇回城市了，他没有挽留——但年糕叼来了一封信，信上只有一句话："你走之后，年糕只喝你泡的茶。我也是。"
```

## Phase 4 产出：名场面设计（3个）

### 名场面1：雨中拿铁

```
【类型】甜 ★★★★
【时机】第二幕暧昧期，苏晚棠刚被城里前同事嘲讽"混得这么惨"
【场所】古镇石桥上，傍晚暴雨

【场景】
苏晚棠淋着雨坐在桥栏上，假装在拍素材，其实在发呆。
陆时安撑伞过来，没有问她怎么了。
只是把一杯温热的拿铁递过去，伞倾向她那边，自己的半边肩膀淋湿了。

【核心对白】
苏晚棠："你怎么知道我在这？"
陆时安："年糕告诉我的。"
苏晚棠："猫又不会说话。"
陆时安："……嗯。"（停顿）"但它会往你在的方向跑。"

【传播点】
观众会截图："但它会往你在的方向跑。"
观众会说："他说的是年糕，但其实说的是自己。"
```

### 名场面2：灯会走失

```
【类型】甜 ★★★★★
【时机】第三幕表白前夜，古镇百年灯会
【场所】万盏花灯的河道旁

【场景】
人山人海中两人走散。苏晚棠踮脚找他，河灯从脚下漂过。
突然她的花灯亮了——是他刚才偷偷在她的灯上多写了一行字。
她低头看：上面写着"我在你身后"。
她猛地转身，他就站在那里。花灯的光映在他脸上。
他难得笑了。

【核心对白】
苏晚棠："你到底从什么时候开始跟着我的？"
陆时安："从你走散的那一秒。"
苏晚棠："那你为什么不叫我？"
陆时安：（很久的沉默）"……因为我想看你找我。"
```

### 名场面3：车站不告别（虐转甜）

```
【类型】先虐后甜 ★★★★★
【时机】第四幕考验期末尾→第五幕开始
【场所】古镇长途汽车站

【场景】
苏晚棠决定回城市（逃跑本能发作），拖着行李箱到车站。
陆时安没有来送她。
车来了，她上了车，坐在靠窗的位置。
车开动——她看到候车亭的柱子后面，年糕蹲在那里，脖子上挂着一个小纸卷。
她让司机停车，跑下去蹲在年糕面前，展开纸条：
"我不会追你。但如果你回来，棠花拿铁永远给你留着。"
她蹲在路边哭了，年糕伸出爪子蹭她的脸。
然后她抬头——陆时安站在三十米外的咖啡馆门口，手里端着一杯拿铁，没有走过来，但也没有走开。
她拖着箱子走向他。

【核心对白】
苏晚棠：（红着眼睛）"这杯冷了。"
陆时安："我重新做。"
苏晚棠："我说的不是咖啡。"
陆时安："……我知道。"
苏晚棠："你为什么不追我？"
陆时安："因为我希望你回来是因为你想回来，不是因为我追你。"
```

## 融合元素交叉设计

| 融合元素 | 和感情线的交叉 |
|---------|-------------|
| **萌宠·年糕** | 年糕是两人的情感信使：叼信件/带路/用猫语吐槽两人"你们够了" |
| **奇幻·猫语术** | 陆时安通过年糕知道苏晚棠的很多真实想法，但不能说出来 → 甜蜜的秘密+考验诚实 |
| **回乡振兴** | 两人合力拯救古镇：她做美食+文旅宣传，他设计改造建筑 → 朝夕相处中日久生情 |
| **事业线** | 苏晚棠的美食视频意外爆火带动古镇旅游 → 城里公司想挖她回去 → 走or留成为感情最大考验 |

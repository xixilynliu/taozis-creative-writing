# 故事灵感引擎 (Story Spark Engine)

> 用标签碰撞生成有记忆点、有话题度的故事灵感。基于8维标签矩阵 × 经典36剧情模型 × 近5年平台爆款数据。

## 这是什么？

一个故事灵感生成器 skill。通过8个维度的标签矩阵强制碰撞，产出有记忆点的一句话概念、有话题度的故事梗概、有辨识度的角色人设。

数据基础：
- **番茄小说**官方37个分类 + 热推排行 + IP金品共创计划热门品类
- **红果/河马短剧** ADX 2025H1 TOP20题材排行
- **爱优腾芒**六大剧场分类 + 年度爆款题材
- **阅文/晋江**销售数据 + 网文一级分类体系
- 中国社科院《2024网络文学发展报告》

## 核心理念

- **反差即记忆点**：标签越不搭，碰撞出来的概念越有新意
- **冲突即话题度**：好的概念天然自带社交讨论基因
- **数据即方向**：用真实平台数据给灵感加上商业判断

## 8维标签矩阵

| 维度 | 解决什么问题 | 标签量 |
|------|-------------|--------|
| 1. 戏剧内核 | 故事要讲什么 | 18个 |
| 2. 题材/世界观 | 故事发生在哪 | 50+标签，按平台热度S/A/B/C分级 |
| 3. 主角身份 | 谁的故事 | 40+标签，含短剧高频人设 |
| 4. 金手指/特殊设定 | 凭什么能赢 | 35+标签 |
| 5. 核心关系 | 和谁的故事 | 12个关系模型 |
| 6. 情绪钩子 | 读者为什么追 | 10种情绪 |
| 7. 话题钩子 | 为什么有传播力 | 25+社会议题 |
| 8. 结构/叙事花招 | 怎么讲 | 10种叙事技法 |

## SOP 概览（5阶段）

| 阶段 | 名称 | 做什么 |
|------|------|--------|
| Phase 0 | 需求诊断 | 确认平台、受众、情绪偏好、已有限制 |
| Phase 1 | 标签碰撞 | 从8维抽标签，碰撞5-8个候选，筛选TOP3 |
| Phase 2 | 概念打磨 | 一句话概念 + 平台适配度 + 话题度预判 |
| Phase 3 | 人设速写 | 记忆点三件套（视觉锚点+反差标签+专属动作） |
| Phase 4 | 梗概生成 | 800字梗概（开篇钩子+核心冲突+3转折+结局方向） |

另有 **Phase 5 批量碰撞模式**：固定1-2个维度，排列组合出10个一句话概念。

## 知识库（5大模块）

| 模块 | 内容 |
|------|------|
| A. 爆款标签组合拆解 | 女频TOP15 + 男频TOP15 + 短剧TOP10 爆款公式 |
| B. 标签碰撞灵感库 | 50个现成组合（女频25+男频15+大众向10） |
| C. 平台适配指南 | 各平台标签权重表 + 短剧专项 + 爆款标题公式 |
| D. 人设记忆点设计公式 | 视觉锚点/反差标签/专属标签命名法 |
| E. 碰撞失败修复指南 | 7种常见问题的急救方案 |

## 安装

```bash
cp -r skills/story-spark ~/.claude/skills/
```

## 使用

安装后直接描述需求：

```
帮我想一个适合番茄女频的故事灵感
```

```
我想写末世题材，但要有新意，帮我碰几个组合
```

```
给我10个一句话概念，要适合改短剧的
```

AI 会按 SOP 逐步引导你完成从标签碰撞到梗概输出的全流程。

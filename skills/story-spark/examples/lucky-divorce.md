## Phase 2 产出：一句话概念打磨

### 概念1：锦鲤离婚

> **一句话**：签完离婚协议走出豪宅那天，她的锦鲤体质觉醒了——前夫的运气全转到了她身上，他的公司股价暴跌，而她随手买的彩票中了五百万。

- 平台适配度：番茄女频 **S** / 红果短剧 **S**
- 话题度预判：**极高**（离婚后逆袭 + 锦鲤 + 前夫后悔，天然热搜体质）
- 差异化：锦鲤不是普通好运，而是"运气转移"——前夫越倒霉她越幸运，爽感成倍放大

### 概念2：穿书炮灰不爱了

> **一句话**：网文作者穿进自己写的虐文，成了那个被男主辜负到死的炮灰女配——这次她按剧情该哭的地方转身就走，男主慌了。

- 平台适配度：番茄女频 **A** / 红果短剧 **A**
- 话题度预判：**高**（穿书+打破第四面墙+追妻火葬场）
- 差异化：主角是"原作者"，她比任何人都知道剧情走向，但每次她改剧情都会触发蝴蝶效应

### 概念3：和婆婆换了身体

> **一句话**：除夕夜和婆婆大吵一架后互换了身体，她得用58岁的身体去跳广场舞、给孙子辅导作业、伺候公公——七天后她哭着说"妈，我错了"。

- 平台适配度：红果短剧 **S** / 快手短剧 **S** / 番茄 **B**
- 话题度预判：**极高**（婆媳关系+身体互换+催泪+沙雕，全年龄段共鸣）
- 差异化：不是"婆婆是恶人"的简单叙事，而是真正的换位理解，结尾治愈

## Phase 3 产出：人设速写（以概念1为例）

### A角色：苏念

```
【一句话人设】离婚那天运气爆棚的前豪门太太

【记忆点三件套】
- 视觉锚点：左手无名指晒痕——摘了婚戒但印记还在
- 反差标签：表面温顺认命，其实内心住着一头狼
- 专属小动作：紧张时数手指，从小指数到大拇指再数回来

【核心矛盾】
- 她最想要的是：证明自己不是"离了谁就活不了"
- 但最害怕的是：运气消失的那天怎么办——她会不会还是那个一无所有的自己
- 撕裂点：运气是底气还是拐杖？

【情感创伤】
- 伤口：结婚七年全职带娃，社会身份只剩"陈太太"，连名字都快忘了
- 保护机制：习惯性说"都行""你决定""我没关系"
- 治愈钥匙：第一次用自己的名字签商业合同的那天

【锦鲤体质设定】
- 规则：她的运气和前夫负相关——她越幸运，前夫越倒霉
- 限制：运气只在她"为自己做决定"时触发，替别人做决定时无效
- 代价：运气不能选择方向——有时候幸运的结果不是她想要的
```

### B角色：陈景行（前夫）

```
【一句话人设】离婚后才发现自己有多离不开她的霸总

【记忆点三件套】
- 视觉锚点：办公桌抽屉里一直放着她做的难吃的曲奇（离婚后也没扔）
- 反差标签：表面控制一切的商业精英，其实是个被运气抛弃后手足无措的纸老虎
- 专属小动作：焦虑时无意识地转无名指上不存在的戒指

【核心矛盾】
- 他最想要的是：让一切回到从前（包括她）
- 但问题是：她已经不是从前那个人了
- 追妻火葬场等级：地狱级——她不恨他，只是不需要他了
```

## Phase 4 产出：梗概

**【一句话概念】**
签完离婚协议那天她的锦鲤体质觉醒了——前夫的运气全转到了她身上。

**【开篇钩子】**
苏念从律所出来，手里攥着离婚协议，对面的陈景行头也不回地上了车。她站在路边等出租车，低头看到地上一张刮刮乐，刮开——五十万。与此同时，陈景行的车胎爆了。

**【核心冲突】**
苏念的每一次好运都对应着陈景行的一次倒霉。她想独立、想证明自己，但运气带来的成功是"她自己的本事"还是"锦鲤体质的赠品"？而陈景行在连续遭遇厄运后终于开始反思——他失去的不只是运气。

**【三个关键转折】**

**转折1**：苏念随手投的简历被三家公司抢着要，同一天陈景行的公司丢了最大客户。她开始怀疑这不是巧合。

**转折2**：她用自己的方案谈下一个大项目，所有人夸她能干——但她发现竞标对手就是陈景行的公司。她赢了，他输了。她第一次不想赢。

**转折3**：陈景行彻底破产。他来找她，不是求复合，是把一个U盘给她——里面是他七年来偷偷录的她和孩子的视频。"这是你的，我一直该给你，但舍不得。"

**【结局方向】**
苏念关掉锦鲤体质（发现方法是"真心为另一个人做一次决定"），用自己的真本事重新开始。陈景行的运气回来了，但他选择不用——从零开始。两个人不是复合，是重新认识。最后一幕：她用自己的名字开了一家小公司，门口碰到他，他说"你好，我叫陈景行。"她笑了："苏念。"

**【记忆点清单】**
- 最爽的场景：前婆婆在她开的店门口排队两小时，进门才认出她
- 最有传播力的台词："你的运气不是锦鲤给的，是你终于开始为自己活了。"
- 最大的反转：锦鲤体质不是奖励——是她过去七年压抑的"运气"一次性释放

# 脑洞基建文·架构师 (Brain Infrastructure Novel)

> 把再疯狂的脑洞，焊死在硬核基建逻辑上。

## 这是什么？

一个专为"脑洞+基建"类网文创作设计的 AI 写作技能。覆盖从类型诊断到章节落笔的全链路，让你不再"脑洞很大但写不下去"。

## 适合写什么？

- 穿越到秦朝搞基建，秦始皇以为你在炼丹
- 克苏鲁世界里用触手铺路、拿SAN值当电费
- 修仙界不打架，专门盖楼修路建宗门
- 末世废土从零重建文明
- 任何"从一片荒地变成繁华都市"的故事


## SOP 概览（8 阶段）

| 阶段 | 名称 | 做什么 |
|------|------|--------|
| Phase 0 | 类型诊断 | 三问定性 → 确认是不是基建文、属于哪个亚种 |
| Phase 1 | Input 采集 | 收集四大参数：脑洞 / 风格 / 金手指 / 初始环境 |
| Phase 2 | 设定锚定 | 把脑洞转化为可采集、可加工、可消耗的资源 |
| Phase 3 | 金手指设计 | 明确能力边界、绘制科技树、设计采集链 |
| Phase 4 | 建设节奏 | 三阶段（生存→生活→生产）× 痛点/方案/爽点/NPC反应 |
| Phase 5 | 迪化反应堆 | 利用信息差设计误解场景，制造爽点/笑点 |
| Phase 6 | 危机升级 | 建设行为的因果链副作用 → 基建式解法（不是打架） |
| Phase 7 | 章节落笔 | 单章结构模板、五感描写、对白规则、章间衔接 |

阶段之间可回环迭代——危机解决后发现新资源，回到 Phase 2 重新锚定。

## 安装

```bash
cp -r skills/brain-infra-novel ~/.claude/skills/
```

## 使用

安装后直接对话即可自动触发，或手动调用：

```
我想写一个脑洞基建文，设定是[你的脑洞]
```

AI 会按 SOP 逐步引导你完成从设定到正文的全流程。

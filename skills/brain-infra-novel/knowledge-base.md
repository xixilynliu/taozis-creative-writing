## 模块 A：硬核基建逻辑 (Realism Anchor)

哪怕写魔法世界，这些逻辑也不能丢，否则会飘。

### A1. 物资流转逻辑

| 需求 | 产业链 |
|------|--------|
| **吃** | 农业（育种→肥料→灌溉）→ 食品加工（储存→防腐） |
| **穿** | 纺织（原料→纺纱→织布→印染） |
| **住** | 建材（采集→烧制→搭建）→ 供暖/制冷 |
| **行** | 道路（勘测→路基→路面）→ 载具（畜力/机械） |

**关键**：产业链。不要只写造出了汽车，要写为了造汽车，先得建炼钢厂，为了炼钢，先得挖煤矿。

### A2. 社会组织逻辑

| 维度 | 要解决的问题 |
|------|-------------|
| **分工** | 谁负责种地？谁负责打铁？谁负责记账？ |
| **管理** | 如何分配食物？平均主义还是按劳分配？ |
| **法律** | 偷东西怎么判？杀人怎么判？ |
| **货币** | 用什么当钱？贝壳、金币、还是"贡献点"？必须有一般等价物 |

### A3. 科技树层级

| Tier | 代表技术 | 文明阶段 |
|------|---------|---------|
| Tier 0 | 石器/木器 | 原始 |
| Tier 1 | 陶器/青铜器/初级魔法 | 部落 |
| Tier 2 | 铁器/水泥/蒸汽机/中级魔法 | 封建/工业早期 |
| Tier 3 | 电力/化工/内燃机/高级魔法 | 工业/现代 |
| Tier 4 | 核能/信息技术/规则修改 | 未来/神话 |

**注意**：主角必须一级一级升，不能跳跃太快，否则缺乏成就感。

## 模块 B：脑洞混搭模板 (Brainhole Mixer)

随机组合以下维度，生成新颖设定：

### B1. 基建要素替换表

| 维度 | 选项A (传统) | 选项B (怪异/科幻) | 选项C (克苏鲁/修仙) | 混搭示例 |
|------|-------------|-------------------|---------------------|---------|
| **能源** | 煤炭/石油 | 核能/太阳能 | 灵魂/SAN值/信仰力 | 用"克苏鲁的疯狂值"作为核能发电的冷却剂 |
| **建材** | 砖石/木材 | 钢铁/玻璃 | 骨骼/血肉/规则符文 | 用"历史的尘埃"压缩成的板块盖楼 |
| **动力** | 人力/畜力 | 蒸汽/电力 | 御剑/念力/触手 | 千万只蚂蚁组成的生物计算机控制的机械臂 |
| **通讯** | 信鸽/驿站 | 电话/电报 | 心灵感应/托梦 | 通过寄生真菌连接所有人的意识网络 |

### B2. 世界观模板快速生成

**公式**：`[反常识物理规则] + [现实基建需求] = 脑洞基建`

示例：
- "声音是实体" + 建筑 = 必须在静音环境下施工
- "影子是液体" + 能源 = 用影子做燃料发电
- "记忆可以物质化" + 建材 = 用回忆盖房子，住进去能看到房主的一生
- "时间有形状" + 交通 = 把"昨天"的路延伸到"明天"的城市

## 模块 D：NPC反应模板

### D1. 标准"真香"五阶段

| 阶段 | 情绪 | 典型台词 |
|------|------|---------|
| 1 震惊 | 不敢相信 | "这……这怎么可能！" |
| 2 恐惧 | 认为是邪术 | "妖法！快跑！" |
| 3 试探 | 好奇心战胜恐惧 | "我能……摸一下吗？" |
| 4 真香 | 享受成果 | "呜呜呜，活了这么久第一次……" |
| 5 崇拜 | 完全臣服 | "神使大人！请受我一拜！" |

### D2. 高级版：不同身份的NPC反应差异

| NPC类型 | 反应特点 | 示例 |
|---------|---------|------|
| **老者/长老** | 用古老传说解释一切 | "预言中说过，'铁鸟'降临之日……" |
| **战士/将军** | 第一反应是能不能打仗用 | "这'火雷'能不能装到弩上？" |
| **祭司/巫师** | 认为触犯了神灵 | "你在亵渎大地之母！" |
| **商人** | 第一反应是能不能卖钱 | "这'透明石'（玻璃），一块能换几头牛？" |
| **小孩** | 毫无恐惧，纯粹好奇 | "叔叔，这个亮亮的东西能吃吗？" |
| **统治者** | 想据为己有或控制你 | "仙长，愿不愿意做寡人的国师？" |

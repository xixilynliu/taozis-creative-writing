# 示例：穿越大秦搞基建（搞笑版）

> 风格：历史系 + 工业魔改 + 迪化流 + 脑残广告风

---

## 设定概要

- **核心脑洞**：穿越到秦朝，系统是个只会发"拼多多砍一刀"链接的烂尾工程助手
- **基建风格**：历史系 + 工业魔改 + 脑残广告风
- **金手指**："拼多多基建系统"，所有图纸都需要"邀请好友砍一刀"才能解锁，砍价永远差0.01%
- **核心冲突**：你想建现代化厕所，秦人以为你在炼"长生不老丹"；你想修水泥路，秦始皇以为你在布"锁龙阵"

---

## 阶段一产出：《大秦土特产-工业垃圾对照表》

### Query
```
【设定锚定】请基于"秦朝"背景，填写《土特产-工业垃圾对照表》：
- 秦砖汉瓦：不要当建材，要当成什么高科技耗材？
- 兵马俑：不要当文物，要当成什么自动化设备？
- 长生不老药（水银）：不要当毒药，要当成什么清洁能源？
- 长城：不要当防御工事，要当成什么巨型机器？
要求：必须有"强行解释"的逻辑，越一本正经胡说八道越好。
```

### Sample Output

| 土特产 | 工业魔改用途 | 应用 |
|--------|------------|------|
| **秦砖** | 高密度数据存储盘 | 质地坚硬，刻上二进制代码（用凿子刻0和1），防水防火防虫蛀，天然"冷数据存储中心"。主角建立"阿房宫数据中心"，用秦砖存《秦律》和小黄文 |
| **兵马俑** | 3D打印生物耗材 | 陶土外壳+竹子液压杆+齿轮+特制胶水 = 不知疲倦的"秦式高达"。缺点：下雨会泥化，必须打伞 |
| **水银** | 超导液/地暖循环液 | 秦始皇陵的水银江河改造成"全城地暖循环系统"。虽然有剧毒，但只要不喝，取暖效果拔群，还能杀菌 |
| **长城** | 巨型服务器机柜/信号基站 | 烽火台安装铜镜反射太阳光，白天当信号塔，晚上当灯塔，兼职给阿房宫供电（太阳能版） |

## 阶段二产出：金手指限制与采集链

### Query
```
【金手指限制】主角的金手指叫"拼多多基建系统"，所有图纸都需要"邀请好友砍一刀"才能解锁，而且砍价永远差0.01%。
请设计一个"采集水泥原料"的链条：主角需要烧水泥，但秦朝没有石灰石，系统提示"去骊山挖秦始皇的祖坟石"，挖开会触发什么搞笑的惩罚？
```

### Sample Output

1. **工具制作**：主角想烧水泥，系统提示【水泥配方：石灰石+粘土】。系统地图显示最近的石灰石在骊山地宫（秦始皇还没死，正在建）。
2. **材料升级**：不能明挖，只能偷偷"借"。用"洛阳铲"打洞，结果挖出来的不是石头，是"秦代手办（兵马俑半成品）"。
3. **操作过程**：主角把手办砸碎磨成粉，混合粘土烧出了"劣质水泥"。
4. **搞笑惩罚**：因为砸了秦始皇的"手办"，系统触发了【祖龙诅咒】——只要主角一修房子，方圆十里内的青蛙就会集体叫春，噪音污染导致无法睡觉。主角必须先发明"隔音耳塞"或者"青蛙诱捕器"才能继续施工。

## 阶段三产出：建设三阶段（搞笑版）

### Query
```
【建设三阶段】针对"大秦朝"，请细化三个阶段：
- Level 1 生存：如何解决"上厕所没有纸且有蛆"的问题？
- Level 2 生活：如何解决"冬天冷得像冰柜且没暖气"的问题？
- Level 3 生产：如何解决"运粮太慢且路太烂"的问题？
要求：必须有"秦人把主角当神仙/疯子"的具体反应。
```

### Sample Output

**Level 1 生存基建：自动冲水茅房**（被误认为"化粪池阵法"）

- **建设**：三格化粪池，陶管连接，虹吸原理拉绳冲水。
- **爽点**：主角拉绳，"哗啦"一声，便便瞬间消失。
- **秦人反应**：隔壁老王吓尿了，以为主角在茅房里养了"吞噬万物的妖兽"。每次主角上厕所，全村人都要在外面磕头，求妖兽别跑出来吃人。

**Level 2 生活基建：地暖火墙**（被误认为"炼丹房炸炉"）

- **建设**：水银封在竹管里埋在墙脚下，烧锅炉加热水银循环。
- **爽点**：外面大雪纷飞，屋里穿短袖吃西瓜。
- **秦人反应**：扶苏公子来视察，一进屋热得脱衣服，震惊道："此屋竟有地火之气！且无烟无味！先生莫非在屋内炼制'寒暑不侵丹'？" 主角："对对对，这是五毒散炼出来的，你要不要尝尝？" 扶苏："不了不了。"

**Level 3 生产基建：秦直道**（被误认为"天轨"）

- **建设**：水泥+大石头铺笔直硬路，画白线，立"左侧通行"木牌。
- **爽点**：马车跑到60码不颠簸，运粮效率提升10倍。
- **秦人反应**：六国刺客马车太快冲出路面翻车。刺客头目爬出来看着水泥路崩溃大哭："这路太滑了！根本借不到力！这是妖术！是专门克制我们轻功的妖术！"

## 阶段四产出：迪化反应（巨猪事件）

### Query
```
【迪化设计】主角只是为了"给猪饲料里加点抗生素"，结果猪吃了之后长得像大象一样大，且不生病。
请设计秦始皇的误解链条：
- 他看到了什么（巨型猪）
- 他联想到了什么（长生不老/兵马俑强化）
- 他给主角封了什么离谱的官职
```

### Sample Output

**身体变化**：这猪不仅大（两千斤），而且眼神清澈（其实是吃药吃傻了），一拳能打碎石磨。

**错误归因**：秦始皇来视察，倒吸一口凉气："此猪……竟有上古凶兽'并封'的血脉！但这血脉太过纯净，毫无杂质！赵高！这定是仙家手段洗去了凡胎！"

**迪化高潮**：秦始皇激动地抓住主角的手："仙长！您是不是已经参透了'肉白骨、活死人'的奥秘？这猪是不是吃了您的'太上老君特供金丹'？寡人不求长生，只求您把这'金丹'给朕的百万大军服用！朕要造一支'巨猪军团'！不，是'神象军团'！"

**主角反应**：看着手里的抗生素空瓶，冷汗直流："陛下，这药有副作用，吃多了会肾衰竭……啊不，会折寿。而且这猪还没阉割，脾气暴躁。"

**结局**：秦始皇大手一挥："无妨！给猪也封个'天蓬大元帅'！即刻起，全国搜罗种猪，交由仙长调配！"

## 阶段五产出：危机升级（山寨维权+交通肇事）

### Query
```
【危机设计】因为主角修了"秦直道（水泥路）"，导致六国旧贵族马车太快，频繁超速追尾。同时，因为主角用"兵马俑"当劳动力，真正的兵马俑（有灵魂的）觉得自己被"山寨"了要来维权。
请描述主角如何用"基建手段"解决这场危机？
```

### Sample Output

**危机爆发**：
- 咸阳街头，六国贵族马车在水泥路上飙车，连环追尾，"咸阳大堵车"
- 骊山方向传来巨响，几千个真·兵马俑活过来了拿着戈矛找主角："你个奸商！用陶土做的劣质品冒充我们！还我们肖像权！"

**威胁**：前面是愤怒的陶俑大军，后面是堵了十里的马车长龙。秦始皇在阿房宫顶上拿着望远镜看，准备把主角扔下去祭天。

**基建解法**（"交通管制+品牌联名"方案）：
1. 主角当场拿出红漆，在所有山寨兵马俑身上画上"真·皇家认证"的Logo，给每个人发一顶"安全帽"（竹编）
2. 宣布"皇家特许经营"：只有真·兵马俑才能上秦直道当"交警/收费员"
3. 发明"红绿灯"（不同颜色的灯笼），制定《大秦交通法》：超速者罚款"一甲"（相当于扣12分）

**结局**：真·兵马俑一听能当"官"还能收过路费，立刻不闹了，高兴地穿上反光背心（铜片做的）去指挥交通。秦始皇感叹："仙长真乃经世之才！连鬼魂都能为朕所用！这'收费站'的主意，甚妙！甚妙！"

## 核心笑点公式总结

```
现代常识（理所当然） + 古代迷信（过度解读） + 主角心虚（强行圆谎） = 爆笑
```

| 主角做的事 | 秦人理解 | 笑点来源 |
|-----------|---------|---------|
| 修化粪池 | 养了吞噬妖兽 | 恐惧+崇拜 |
| 装地暖 | 炼丹房地火 | 过度解读 |
| 铺水泥路 | 克制轻功的妖术 | 敌人视角 |
| 喂猪抗生素 | 太上老君金丹 | 皇帝视角 |
| 搞红绿灯 | 天道轮回显化 | 正经胡扯 |

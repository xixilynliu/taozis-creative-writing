# 示例：重力是液体的世界

> 风格：蒸汽朋克 + 物理规则怪谈（正经硬核版）

---

## 设定概要

- **核心脑洞**：重力像水一样会流动、有潮汐，甚至能像石油一样被抽取。低洼处重力深不见底（会被压死），高处重力稀薄（会飘走）。
- **基建风格**：蒸汽朋克 + 物理规则怪谈
- **金手指**："牛顿的苹果核"——只能检测重力流向，不能直接改变重力
- **初始环境**：重力潮汐频繁的危险荒野

---

## 阶段一产出：《异化资源-现实功能对照表》

### Query
```
【设定锚定】请基于"重力是液体"的设定，填写《异化资源-现实功能对照表》：
- 重油（高密度重力液体）：对应什么建材功能？
- 重力潮汐：对应什么能源功能？
- 失重气泡：对应什么工业功能？
要求：必须符合物理逻辑，不能凭空变出反重力。
```

### Sample Output

| 资源 | 学名 | 现实功能 | 应用 |
|------|------|---------|------|
| **重油** | 高密度重力液 | 超级混凝土/地基材料 | 普通泥土会被重力压散，但注入"重油"搅拌后的混凝土，密度极大，能像船浮在水上一样"浮"在重力场中，不会被压塌 |
| **重力潮汐** | 引力势能波 | 水力发电机/液压机 | 不需要烧煤，只要安装"重力涡轮机"，潮汐来临时巨大的重力压差直接驱动齿轮旋转，带动工厂机器 |
| **失重气泡** | 引力真空泡 | 起重机/物流运输 | 收集重力潮汐退去后留下的"真空泡"，填充到气囊里就是天然的吊车，可以把几吨重的石块轻松吊起 |

## 阶段二产出：金手指限制与采集链

### Query
```
【金手指限制】主角的金手指是"牛顿的苹果核"（只能检测重力流向），不能直接改变重力。
请设计一个"从采集到成品"的链条：主角如何利用这个苹果核，在充满危险的重力流中采集"重油"？
要求：必须有具体的工具制作过程和危险。
```

### Sample Output

**采集链条**：

1. **工具制作**：主角用苹果核检测到"重油"沉淀在沼泽下30米（那里重力是地面的10倍）。普通铁桶会瞬间压扁。
2. **材料升级**：主角必须先猎杀"抗压兽"，用它们的皮做成袋囊，再混入"固化沙砾"制成"抗压采集袋"。
3. **操作过程**：主角不能亲自下去（会被压成肉饼），必须制作一个"重力滑轮组"，利用失重气泡的浮力，像钓鱼一样把装满重油的皮袋钓上来。
4. **代价**：每次提取重油，滑轮组都会发出刺耳的摩擦声，容易引来以"震动"为食的"音爆虫"。

## 阶段三产出：建设三阶段

### Query
```
【建设三阶段】针对"重力液体"世界，请细化三个阶段的核心痛点和标志性事件：
- Level 1 生存：如何解决"睡觉时重力变大把人压死"的问题？
- Level 2 生活：如何解决"上厕所时重力反向把人冲上天"的问题？
- Level 3 生产：如何利用"重力差"建立交通网？
要求：每个阶段必须有一个"第一次成功"的爽点细节。
```

### Sample Output

**Level 1 生存基建：锚定床（The Anchor Bed）**

- **痛点**：晚上重力潮汐来袭，睡在地上会被压进地底。
- **建设**：主角发明"重力锚"。用链条把床锁在地下的岩石层，链条上安装弹簧液压杆来缓冲重力变化。
- **爽点细节**：深夜，重力瞬间增加3倍，普通人的床腿折断，但主角的床只是微微下沉，弹簧发出令人安心的"嗡"声，主角安睡无恙。原住民第一次知道"睡觉不用担心被压扁"。

**Level 2 生活基建：单向阀厕所**

- **痛点**：排泄物会因为重力紊乱飞溅。
- **建设**：建立"重力虹吸管道系统"。利用地下的高密度重油产生强引力，像黑洞一样把所有东西吸走，只进不出。
- **爽点细节**：第一个用户上完厕所，惊恐地发现排泄物瞬间消失，且没有任何异味上返，跪地痛哭："这是通往地心的净化之口！"

**Level 3 生产基建：重力滑梯高速路**

- **痛点**：运输货物太重，畜力拉不动。
- **建设**：在城市间挖掘"重力槽"。利用两地的重力势能差（A地重力1.5G，B地0.5G），让货物放入滑槽后，靠重力落差自动滑向B地。
- **爽点细节**：第一批货物（几十吨矿石）没有用任何马力，仅仅是解开了刹车，就在重力牵引下以100km/h的速度呼啸而过，尘土飞扬。

## 阶段四产出：迪化反应

### Query
```
【迪化设计】主角只是为了"给工厂锅炉降温"，不得不把一条地下的"重力暗河"引到地表，导致周围形成了一个"低重力场"，大家走路轻飘飘的。
请设计原住民（一群重力过大的矮人）的误解链条：
- 他们的身体变化
- 他们的错误归因
- 他们给主角起的称号
```

### Sample Output

**身体变化**：老矮人"铁骨"原本因为关节承受不了自身重量（300斤），已经瘫痪十年。走进白雾区域后，发现膝盖不痛了，甚至轻轻一跳就能蹦起一米高。

**错误归因**：铁骨惊恐又狂喜地摸着地面："这不是风……这是'神的恩赐'！传说中只有死后的灵魂才能摆脱重力的枷锁！主人……主人他在让我们提前享受死后的世界！"

**迪化高潮**：铁骨跪在主角脚边，亲吻主角沾满机油的靴子："伟大的'引力解放者'！您是要把我们从大地的诅咒中救赎出来吗？请允许我们为您献上最重的矿石，只求您让这'神圣的微风'多吹一会儿！"

**主角反应**："呃，其实那是废气排放口，而且这风吸多了会不孕不育……算了，先忽悠他们把矿石搬过来再说。"

**称号**：引力解放者 / 无重之神

## 阶段五产出：危机升级

### Query
```
【危机设计】因为主角在Level 3阶段修了"重力滑梯高速路"，过度抽取了地下的"重力油"，导致地壳下方出现了一个巨大的"重力空洞"。
请描述：
- 这个空洞引发的自然现象
- 如何威胁到主角刚建好的城市
- 主角必须用什么"基建手段"来解决？
```

### Sample Output

**现象**：城市中心广场突然出现了"反重力喷泉"。不是水喷出来，而是广场本身开始缓缓升空！因为地下重力油被抽干，底层变成了"负重力层"，像气球一样要把城市带飞。

**威胁**：如果不加控制，3天后整个城市会飘到平流层，所有人冻死或窒息。原本的"重力锚定床"失效，大家睡觉会飘在天花板上下不来。

**基建解法**：
1. **方案**："压舱物计划"。必须在24小时内向城市地基反方向注入巨量"超重物质"。
2. **执行**：动员全城人去挖掘隔壁"禁地"里的"黑矮星碎片"（密度极大的陨石），用重力滑梯运回来，填入地基底部。
3. **冲突点**：挖掘黑矮星碎片会引来守护碎片的"引力虫"，主角必须边打边修，在虫群中建立临时"传送带"。

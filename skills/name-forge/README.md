# 命名锻造炉 (Name Forge)

> Status: **Ready**

## 简介

命名锻造炉。从故事梗概和风格出发，一次性输出角色名、地名、组织名、道具名、书名/剧名的完整命名方案。

核心理念：**好名字不是好听，是自己会讲故事。**

## 特色

- **4条命名路径**：风格驱动、压缩叙事、反差制造、系统构建
- **6种记忆点技巧**：反差、缺失感、通感错位、动势感、语义叠加、口感上瘾
- **顶流IP验证**：从琅琊榜、三体、盗墓笔记、全职高手、哑舍、甄嬛传等顶级IP中提取命名规律
- **全套输出**：人名+地名+组织名+道具名+书名，一次性搞定
- **口感校验**：声调搭配、声母错开、简称预判

## 结构

```
name-forge/

```

## 安装

```bash
cp -r skills/name-forge ~/.claude/skills/
```

## 使用

直接说"帮我起名"，或在 story-spark 生成梗概后自动衔接。

最小输入只需要：
1. 故事梗概或一句话概念
2. 风格关键词（搞笑/文艺/燃/虐/悬疑/史诗）
3. 需要命名的对象清单

# 女频耽美文·写作技巧

> task · novel — 情感暗涌、张力构建、关系升温

## 一、心动书写

- **「不经意」时刻**：不小心碰到手指、目光无意中追随、为对方挡了一下风
- **否认期描写**：「我只是习惯了他在」「任何人都会这样做」——越否认越真实
- **呼吸叙事**：用「呼吸乱了」「忘了呼吸」「刻意平稳的呼吸」传递紧张

## 二、关系升温阶梯

1. 物理距离缩短（从刻意保持距离到自然靠近）
2. 称呼变化（姓→名→昵称/独有称呼）
3. 专属习惯（只为对方做的小事）
4. 身体边界打破（肩膀→手→拥抱→更多）
5. 语言告白（或不说破但彼此都懂）

## 三、虐心与甜蜜并行

- **社会压力**：外部阻力让感情更有重量
- **身份/立场冲突**：明知不该爱但控制不住
- **甜中带刀**：最甜蜜的时刻埋下最大的隐患

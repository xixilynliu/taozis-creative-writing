# 升级流文·写作技巧

> task · novel — 成长曲线、战斗爽感、境界设计

## 一、成长弧线

- 每个修炼阶段有独特挑战：不是简单重复「修炼→突破」
- 瓶颈期是故事好素材：焦躁、自我怀疑、寻找灵感
- 机缘获取要有代价或选择：选A放弃B，有取舍才有故事

## 二、战斗爽感

- **「展示新技能」型战斗**：突破后第一战让读者看到质变
- **以弱胜强的关键在「策略」**：地形利用、弱点分析、团队配合
- **连胜递进**：打完小头目→中层→大BOSS，一口气不断
- **败北也是故事**：输了怎么爬起来比赢了更有看头

## 三、境界叙事

- 每个大境界改变角色的「视野」——不只是力量变化，还有认知升级
- 突破时刻要有仪式感：天地异象、领悟感言、周围人的反应
- 力量对比可视化：用具体行为（一拳碎山、一步千里）替代抽象数字
